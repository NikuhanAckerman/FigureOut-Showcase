# Guia de instalação (PT-BR)

* 1 - Clone o repositório para sua máquina localmente;
* 2 - Inicie o MySQL Server na porta 3306, usando services.msc, XAMPP, WAMPP, ou afins;
* 3 - Crie o banco "figureout" e sua respectiva conexão, usando MySQL Workbench ou afins;
* 4 - Ajuste a senha e o username no application.properties de acordo com sua senha e username do MySQL;
* 5 - Compile a aplicação pelo FigureOutApplication.java em "\figureout\src\main\java\com\project\figureout\FigureoutApplication.java"
* 6 - Caso tudo dê certo, a aplicação vai estar rodando na porta 8080. Busque pela URL localhost:8080 para ver.

# Installation guide (EN)

* 1 - Clone the repository to your local machine;
* 2 - Start MySQL Server at port 3306, either by using services.msc, XAMPP, WAMPP or others; 
* 3 - Create the "figureout" database and its connection, using MySQL Workbench or others; 
* 4 - Change the password and username in application.properties according to your MySQL username and password; 
* 5 - Compile the app by FigureOutApplication.java in "\figureout\src\main\java\com\project\figureout\FigureoutApplication.java" 
* 6 - If everything's done right, the app will start running. Search for localhost:8080 in your browser to check it out.
